IMPLEMENTATION EXAMPLE (SIMPLIFIED)
================================================================================

1. CALL INITIATION:
   ```javascript
   // In your chat screen, add call button
   const initiateCall = async (receiverId) => {
     // Create call document in Firestore
     const callDoc = await addDoc(collection(db, 'calls'), {
       callerId: currentUserId,
       receiverId: receiverId,
       status: 'calling',
       timestamp: Timestamp.now()
     });
     
     // Initialize WebRTC
     const peerConnection = new RTCPeerConnection(config);
     
     // Get local audio stream
     const stream = await mediaDevices.getUserMedia({ audio: true });
     stream.getTracks().forEach(track => {
       peerConnection.addTrack(track, stream);
     });
     
     // Create offer
     const offer = await peerConnection.createOffer();
     await peerConnection.setLocalDescription(offer);
     
     // Send offer to receiver via Firestore
     await updateDoc(callDoc, { offer: offer });
   };
   ```

2. CALL RECEPTION:
   ```javascript
   // Listen for incoming calls
   useEffect(() => {
     const unsubscribe = onSnapshot(
       query(collection(db, 'calls'), 
       where('receiverId', '==', currentUserId),
       where('status', '==', 'calling')),
       (snapshot) => {
         snapshot.forEach(async (doc) => {
           // Show incoming call UI
           showIncomingCallScreen(doc.data());
         });
       }
     );
     return unsubscribe;
   }, []);
   ```

3. CALL ACCEPTANCE:
   ```javascript
   const acceptCall = async (callId, offer) => {
     // Initialize WebRTC
     const peerConnection = new RTCPeerConnection(config);
     
     // Get local audio stream
     const stream = await mediaDevices.getUserMedia({ audio: true });
     stream.getTracks().forEach(track => {
       peerConnection.addTrack(track, stream);
     });
     
     // Set remote description (offer)
     await peerConnection.setRemoteDescription(offer);
     
     // Create answer
     const answer = await peerConnection.createAnswer();
     await peerConnection.setLocalDescription(answer);
     
     // Send answer back via Firestore
     await updateDoc(doc(db, 'calls', callId), {
       answer: answer,
       status: 'active'
     });
   };
   ```
